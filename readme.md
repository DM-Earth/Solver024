# Solver024

This is a simple 24 points solver written in rust.

## Example

```log
1 3 4 5
(1 + 3) + (4 * 5)
1 + (3 + (4 * 5))
3 + (1 + (4 * 5))
3 * (4 - (1 - 5))
3 * (5 - (1 - 4))
4 + (5 * (1 + 3))
```

## Operators

`+, -, *, /`

With super mode:

`>>, <<, ^, &`

## Usage Commands

### Toggle super mode

`super`

### Exit

`exit`, `quit`
